## What is The Valley?

The Valley Inc. is a holding company for a growing portfolio of early-stage startups — each one a standalone product targeting a real, underserved gap. Think Alphabet, but built from scratch.

Every product under The Valley is independently scoped, independently deployed, and grounded in communities that existing tech largely overlooks: commuters on interstate corridors, students at mid-tier universities, local businesses without a digital presence.

This repo is the public-facing landing page — the front door for everything we're building.

## Running Locally

```bash
git clone https://github.com/SrikarSistla08/TheValleyInc.git
cd TheValleyInc
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design System

The site uses a dark minimal aesthetic inspired by Linear and Vercel — intentional, low-noise, and fast.

- **Theme:** Dark background with subtle noise texture overlay
- **Logo:** Custom SVG mountain/valley wordmark
- **Motion:** Scroll-triggered fade-ins, typewriter badge, sticky nav with animated underlines
- **Cards:** Radial glow hover effects
- **Layout:** Responsive grid — 1 → 2 → 4 columns
