# The Valley Inc.

Landing page for [The Valley Inc.](https://thevalleyinc.co) — a portfolio of startups built for students, local businesses, and underserved communities. Designed by [Srikar Sistla](https://www.linkedin.com/in/srikarsistla/).

## Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Fonts:** Inter (body), Instrument Serif (headings)
- **Deployed on Vercel**

## Features

- Dark theme with subtle noise texture
- SVG mountain/valley wordmark logo
- Scroll-triggered fade-in animations
- Typewriter effect on "Building in public" badge
- Sticky nav with animated underline links
- Product cards with radial glow hover effects
- Responsive grid layout (1→2→4 columns)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub and connect to Vercel. One-click deploy.
